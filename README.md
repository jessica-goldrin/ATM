To compile:\
```g++ main.cpp -o atm```\
To run:\
```.\atm.exe```\
Then you'll see the prompt:
```
Hi! Welcome to Jess' ATM Machine!
Please select an option from the menu below.
1 -> Login
2 -> Create Account
3 -> Quit
```
Input ```1``` to login, ```2``` to create an account, or ```3``` to quit the program. You'll be guided through the rest!
